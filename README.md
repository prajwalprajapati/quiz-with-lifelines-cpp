# Quiz with Lifelines System 🎯

A simple interactive quiz game developed in C++ that includes multiple quiz lifelines similar to popular game shows.
The project demonstrates the practical use of C++ concepts like structures, loops, conditional statements, arrays, and file handling.

## 🚀 Features

* Interactive quiz system
* 50-50 Lifeline
* Skip Question Lifeline
* Ask Again Lifeline
* Real-time score tracking
* Quiz history storage using file handling
* User-friendly console interface

## 🛠 Technologies Used

* C++
* File Handling
* Structures
* Loops & Conditional Statements

## 📚 Learning Outcomes

* Programming logic and control flow
* File handling in C++
* Gamification concepts
* Problem-solving and algorithmic thinking
* Modular programming

## 📂 Project Applications

* Beginner C++ mini-project
* Classroom quiz systems
* Learning-based quiz games
* Foundation for advanced quiz applications
